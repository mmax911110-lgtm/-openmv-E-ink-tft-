# 基于 OpenMV 的双屏实时视觉追踪系统

本项目是一款集成视觉追踪、多感光锁定以及双屏数据显示的嵌入式视觉终端。系统通过 OpenMV 捕捉目标色块，并在 TFT 彩屏上提供高速实时预览，同时在 E-ink 墨水屏上输出高对比度的坐标数据。系统具备良好的抗干扰能力和零按键手势交互功能。

## 核心功能

1. 实时坐标追踪。利用 OpenMV 快速定位场景中的特定色块，并在 TFT 屏幕上实时绘制追踪准星。

2. 传感器参数锁定。系统在检测到目标 300ms 后会自动锁定白平衡（AWB）和自动曝光（AEC），有效防止因环境光线突变或反光导致的追踪目标丢失。

3. 异步驱动机制。针对墨水屏刷新缓慢的特性，系统采用了非阻塞驱动逻辑，确保墨水屏在进行全局或局部刷新时，TFT 彩屏的帧率不受明显影响。

4. 无人机级交互逻辑。用户仅需用手遮挡镜头 2 秒（模拟遮光罩），即可在预置的红色、绿色、蓝色追踪目标间循环切换，无需物理按键。

## 硬件清单

1. 控制器：OpenMV Cam H7 / H7 Plus。
2. 状态显示：1.54 英寸 Waveshare 墨水屏 (SSD1681 驱动)。
3. 实时预览：OpenMV 官方支持的 TFT 扩展屏或 SPI 彩屏。
4. 目标色块：标准的红色、绿色、蓝色物体。

## 详细引脚定义

墨水屏与 OpenMV 的接线对应如下：

- Eink SCLK (SCL) -> OpenMV P0
- Eink MOSI (SDA) -> OpenMV P1
- Eink CS -> OpenMV P4
- Eink DC -> OpenMV P5
- Eink RST -> OpenMV P9
- Eink VIN -> OpenMV 3.3V
- Eink GND -> OpenMV GND

系统软件已针对以上引脚进行配置，接线完成后即可运行。

## 使用指南

1. 初始化与搜索模式。启动后，TFT 屏幕将显示实时预览内容，墨水屏显示全屏“SEARCHING”背景。此时系统处于自动感光状态。

2. 锁定与数据显示。当目标进入视野且位置稳定后，系统将自动锁定感光参数。锁定完成后约 0.5 秒，墨水屏会执行一次全局刷新，随后进入局部刷新模式，实时同步显示 X 和 Y 坐标。

3. 目标切换。在任何状态下，完全遮挡摄像头镜头 2 秒，系统会判断为切换指令，自动跳转至下一种颜色目标，并重新开始搜索。

4. 重影消除。为了保持墨水屏的清洁，系统内置了计数器。在搜索状态下，每进行 20 次动画更新会自动触发一轮深度全屏清理刷，防止局部刷新产生的“重影”积累。
